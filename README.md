# tennis

Tennis but with weird physics. Made using Three.js.

## Demo:
https://6471d087ce747362a53ac624--comforting-florentine-9d5cc2.netlify.app/

## Controls:
- R            - Start/Reset

**Player 1:**
- W            - Up
- A            - Left
- S            - Down
- D            - Right
- Spacebar     - Swing / Charge

**Player 2:**
- Up arrow     - Up
- Left arrow   - Left
- Down arrow   - Down
- Right arrow  - Right
- Enter        - Swing / Charge
